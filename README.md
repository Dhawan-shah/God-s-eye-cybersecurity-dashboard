# # 👁 GOD'S EYE – Cybersecurity Monitoring Dashboard

GOD'S EYE is a SOC-style cybersecurity monitoring dashboard built with Python.
It provides visualization and analytics tools for network activity, security events, and threat monitoring.

The goal of this project is to demonstrate the architecture and interface of a modern cybersecurity monitoring platform.

## Features

* Network discovery simulation
* Domain lookup
* Custom port scan interface
* Threat alert monitoring
* Host risk scoring
* Security analytics charts
* Vulnerability heatmap
* Network topology visualization
* Live traffic monitoring graph
* Incident timeline
* Security report generation (PDF)
* Log export (CSV)

## Technologies Used

* Python
* Tkinter
* Matplotlib
* NetworkX
* Seaborn
* ReportLab

## Installation

Install dependencies:

pip install -r requirements.txt

Run the project:

python gods_eye_v17_final.py

## Purpose

This project demonstrates concepts used in Security Operations Centers (SOC) such as:

* network monitoring
* security analytics
* incident visualization
* threat alert management

It is designed as a cybersecurity learning and portfolio project.
