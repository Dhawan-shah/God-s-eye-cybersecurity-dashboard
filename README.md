# 👁 GOD'S EYE – Cybersecurity Monitoring Dashboard

GOD'S EYE is a **SOC-style cybersecurity monitoring dashboard built with Python**.
It visualizes network activity, threat alerts, and security analytics in a command-center style interface.

This project demonstrates the architecture of a **Security Operations Center (SOC) monitoring platform** that combines network insights, threat monitoring, and security analytics in one dashboard.

---

# 🚀 Features

• Network discovery simulation
• Domain lookup tool
• Custom port scan interface
• Threat monitoring alerts
• Host risk scoring table
• Security analytics visualization
• Vulnerability heatmap
• Network topology graph
• Live network traffic monitoring
• Incident timeline tracking
• Security report generation (PDF)
• Security log export (CSV)
• SOC-style dashboard interface

# ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/gods-eye-cybersecurity-dashboard.git
```

Move into the project directory:

```bash
cd gods-eye-cybersecurity-dashboard
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

---

# ▶ Running the Project

Run the dashboard using:

```bash
python gods_eye_v17_final.py
```

After running the command, the **GOD'S EYE Cyber Command Center dashboard will open**.
